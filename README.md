# Zorvyn Finance Dashboard 📊

A clean, interactive, and responsive personal finance dashboard built with React and Vite. This application allows users to track their financial activity, view dynamic spending insights, and manage transactions seamlessly.

## ✨ Features

* **Overview & Insights:** Visualize financial data with dynamic charts (Balance Area Chart, Spend Analysis Pie Chart, and Category Trend Stacked Bar Chart).
* **Transaction Management:** Easily view, add, and delete transactions. Includes real-time category-based search and filtering.
* **Role-Based Access Control (RBAC):** Toggle between 'Admin' (can add/delete transactions) and 'Viewer' (read-only access) simulation.
* **Smart Observations:** Auto-generated AI-style insights based on spending habits (e.g., month-over-month comparisons and top spending categories).
* **Data Export:** Export your filtered transaction history to a CSV file with a single click.
* **Data Persistence:** Transactions are saved locally using the browser's `localStorage` so your data is retained across sessions.
* **Modern Dark UI:** A sleek, polished dark interface enhanced with smooth Framer Motion animations.

## 🛠️ Tech Stack

* **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Charts:** [Recharts](https://recharts.org/)
* **Icons:** [Lucide React](https://lucide.dev/)
